## 이 문제...
-> 대각선을 어떻게 구할 수 있는지 알면 쉽게 풀수있는문제다.

https://leetcode.com/problems/maximum-area-of-longest-diagonal-rectangle/?envType=daily-question&envId=2025-08-26 

여기서 볼수 있구...

```java
class Solution {
    public int areaOfMaxDiagonal(int[][] dimensions) {
        int bestDiag2 = -1;   // 대각선 제곱의 최대값
        int bestArea  = 0;    // 그때의 넓이

        for (int[] d : dimensions) {
            int l = d[0], w = d[1];
            int diag2 = l * l + w * w; // 루트 생략, 제곱합으로만 비교
            int area  = l * w;

            if (diag2 > bestDiag2) {
                bestDiag2 = diag2;
                bestArea = area;
            } else if (diag2 == bestDiag2 && area > bestArea) {
                bestArea = area;
            }
        }
        return bestArea;
    }
}
```
